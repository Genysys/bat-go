## Requirements
 * Docker-Compose 1.24.*


## Instructions 

 * From the root of the directory, run `docker-compose up`.
 * (_Optional_) The command above displays container logs on `stdout`. If this is undesired, you run docker in detached mode with `docker-compose up -d`.
